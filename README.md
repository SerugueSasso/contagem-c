# Contagem de Números em C

Este projeto foi desenvolvido como exercício de programação em linguagem C.

## Descrição do Problema

O programa deve:

* Receber um número inteiro **positivo e maior que zero**
* Imprimir a contagem de **x até x+9** (incluindo ambos)
* Exibir cada valor em uma nova linha utilizando `printf`

---

## Lógica da Solução

1. O usuário informa um número inteiro `x`
2. O programa calcula o limite final (`x + 10`)
3. Um laço `while` é utilizado para imprimir os valores de `x` até `x + 9`
4. Cada número é exibido em uma nova linha

---

## Conceitos Utilizados

* Entrada de dados com `scanf`
* Estrutura de repetição `while`
* Operações com inteiros
* Saída com `printf`
